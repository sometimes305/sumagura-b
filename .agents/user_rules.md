# スマグラ開発ルール

このファイルはスマグラプロジェクトにおけるAIアシスタントの行動指針です。

## 共通ルール
- **バージョン管理**: 
  - 修正やコミットを行うたびに、タイトルメニュー横のバージョン番号をインクリメント（1増やす）すること。
  - 対象箇所：
    - `index.html` の `id="version-display"` 内のテキスト
    - `js/main.js` の `window.SMA.VERSION` および `window.SMA.ID_PREFIX`
- **コミットとプッシュ**: 
  - 修正のたびにコミットを行うが、**自動でプッシュは行わないこと**。
  - プッシュはユーザーから明示的な指示（「プッシュして」など）があった場合のみ実行する。
- **使用言語**: 
  - ユーザーへの応答、コードコメント、ドキュメントは原則として **日本語** で記述すること。
- **コミットメッセージ**: 
  - コミットメッセージの冒頭に現在のバージョン番号（例：v423: ...）を付与すること。
